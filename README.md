# Data-Mesh-on-Google-Cloud

1. Autoryzacja
   
   Przed odpaleniem `terraform apply` musiałam potwierdzić logowanie

   `gcloud auth application-default login`

   Po potwierdzeniu dostaję info, gdzie credentiale zostały zapisane i aktualizuję ścieżkę w main.tf

   Credentials saved to file: [/tmp/tmp.9iER9f9Pde/application_default_credentials.json]
